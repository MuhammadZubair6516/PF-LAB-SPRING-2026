# 📘 CL1002 – Programming Fundamentals Lab  
## 🧪 Lab: Recursion, Structures & File Handling in C

---

## 🎯 Aims

- To understand and implement **recursion** (types & stack calling) in C.
- To explore **constant variables** for fixed values and **static variables** for state retention.
- To learn and apply **structures** for organizing related data.
- To understand and implement:
  - **Nested structures**
  - **Structure composition**
  - **Arrays of structures**

---

## 📌 Objectives

- 🔁 Explore recursive functions and understand how functions call themselves to solve complex problems.
- 🔒 Understand the role of **constants** and **static variables** in managing scope and immutability.
- 🧱 Learn how to define and use **structures** to represent complex data types.
- 🧩 Implement **nested structures** to model hierarchical data.
- 🏗️ Apply **structure composition** for modular and organized data design.
- 📊 Use **arrays of structures** to efficiently manage collections of related data.
- 💾 Implement **file handling** for reading and writing data, enabling persistent storage.
- 👉 Practice accessing and manipulating structure members using:
  - Pointers  
  - Array indexing  

---

## 📖 Introduction

This lab focuses on core programming concepts in C that are essential for building efficient and structured applications. The topics covered help in understanding both **control flow** and **data organization**, which are fundamental for software development.

---

## 🧠 Topics Covered

1. 🔁 Introduction to Recursion  
2. 🧱 Introduction to Structures  
3. 🧩 Nested Structures  
4. 🔒 Constants and Static Variables  

---

## 🛠️ Technologies Used

- Language: **C**
- Compiler: GCC / Any standard C compiler
- IDE: Code::Blocks / Dev-C++ / VS Code

---

## 🚀 Learning Outcomes

After completing this lab, you will be able to:

- Implement recursive solutions effectively  
- Manage data using structures and nested structures  
- Use constants and static variables appropriately  
- Handle files for storing and retrieving structured data  
- Work with complex data using arrays and pointers
