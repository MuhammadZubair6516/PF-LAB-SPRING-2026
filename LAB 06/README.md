# Lab 06 – Introduction to Iterative Structures (Loops)

## Programming Fundamentals – Spring 2026


## INTRODUCTION
In this lab, we study **Iterative Structures (Loops)** in C programming.  
Loops allow a block of code to execute repeatedly until a specific condition is satisfied.  
They are essential for writing efficient and dynamic programs.

---

## TOPICS COVERED
- Introduction to Loops and Their Types  
- For Loop  
- While Loop  
- Do-While Loop  

--- 

---

## AIMS

### 1. Automating Repetitive Tasks
- Eliminate manual repetition by automating repeated tasks.  
- Make programs more efficient and easier to manage.

### 2. Efficient Code Execution
- Reduce the amount of code required for repeated operations.  
- Improve overall program performance and execution speed.

### 3. Improve Code Readability and Maintainability
- Use structured loops instead of duplicated code blocks.  
- Write cleaner, more understandable programs.

### 4. Control Flow Management
- Execute blocks of code repeatedly based on specific conditions.  
- Gain better control over program execution flow.

### 5. Dynamic Input Handling
- Continuously process user input or data sequences.  
- Allow programs to behave flexibly based on changing conditions.

---

## OBJECTIVES

### 1. Iterate Over Data Structures
- Traverse arrays and other collections efficiently.  
- Avoid writing repetitive code for each element.

### 2. Terminate on Specific Conditions
- Ensure loops stop when required conditions are met.  
- Prevent infinite execution.

### 3. Reduce Code Complexity
- Simplify complex tasks using loop structures.  
- Make programs more logical and structured.

### 4. Support Event-Driven Programming
- Enable continuous checking of inputs or triggers.  
- Useful in real-time and interactive applications.

### 5. Flexible Program Execution
- Use control statements like `break` and `continue`.  
- Start, pause, or terminate loops as needed.
