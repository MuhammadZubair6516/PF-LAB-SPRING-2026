# Programming Fundamentals Lab Repository

Welcome to my **Programming Fundamentals (PF) Lab** repository! This repository contains all the tasks and assignments completed during my PF lab sessions. Each lab is organized into separate folders or files, making it easy to navigate and review.

---
## Purpose

The purpose of this repository is to:

- Practice and reinforce core C programming concepts.  
- Maintain organized records of lab tasks for reference.  
- Demonstrate understanding of programming fundamentals, including loops, functions, arrays, pointers, and file handling.

---

## Usage

1. Navigate to the lab folder you want to explore.  
2. Open the `.c` files using any code editor (VS Code, Code::Blocks, Dev C++, etc.).  
3. Compile and run the programs using a C compiler (GCC or similar).
