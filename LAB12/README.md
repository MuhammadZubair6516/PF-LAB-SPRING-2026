# 📘 CL1002 – Programming Fundamentals Lab  
## 🧪 Lab: Dynamic Memory Allocation in C/C++

---

## 🎯 Aims

- To understand the concept of **dynamic memory allocation** in C/C++.
- To apply memory allocation techniques to create, manipulate, and free memory during runtime.

---

## 📌 Objectives

- 💡 Handle **unknown memory requirements** by allocating memory at runtime.
- ⚡ Improve **efficiency** by using memory only when needed.
- 🎯 Gain **flexibility and control** over memory usage and lifetimes.
- 🔄 Allow data to **persist beyond function scope** using dynamic allocation.

---

## 📖 Introduction

This lab introduces the concept of **dynamic memory allocation**, which allows programs to request memory during execution instead of compile time. It is a crucial concept for efficient memory management, especially when dealing with variable-sized data structures.

---

## 🧠 Topics Covered

1. 💾 Introduction to Dynamic Memory Allocation  
2. ⚖️ Static vs Dynamic Allocation  
3. 🛠️ Methods to Allocate Memory in C  

---

## 🛠️ Technologies Used

- Language: **C / C++**
- Compiler: GCC / G++
- IDE: Code::Blocks / Dev-C++ / VS Code

---

## 🚀 Learning Outcomes

After completing this lab, you will be able to:

- Understand the difference between **static and dynamic memory allocation**  
- Use functions like `malloc()`, `calloc()`, `realloc()`, and `free()`  
- Efficiently manage memory during program execution  
- Prevent memory leaks by properly freeing allocated memory  
- Build flexible programs that handle dynamic data sizes
