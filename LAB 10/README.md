# CL1002 – Programming Fundamentals  
## Lab 10: Strings, 2D Character Arrays & File Handling

---

## 📌 Lab Overview

This lab focuses on three important concepts in C programming:

- String manipulation using standard library functions  
- Handling 2D character arrays (arrays of strings)  
- File handling (reading and writing data to files)  

---

## 🎯 Aims

To develop an understanding of:
- String manipulation using string library functions  
- Managing multi-dimensional string data using 2D arrays  
- Performing file operations in C  

---

## ✅ Objectives

- Utilize standard string functions like `strlen`, `strcpy`, `strcat`, `strcmp`  
- Explore 2D character arrays for handling multiple strings  
- Implement file handling techniques using C  

---

## 📖 Topics Covered

- String handling functions (`string.h`)  
- 2D arrays of characters  
- File operations (`fopen`, `fclose`, `fprintf`, `fscanf`, `fgets`)  

---

## 🛠️ Tools & Language

- Language: C  
- Compiler: GCC / Dev-C++ / CodeBlocks
