# Lab 09: Introduction to Functions and Pointers

## AIMS AND OBJECTIVES

### Aims
- To develop an understanding of functions and pointers.

### Objectives
- Familiarize with function declaration, definition, and calling in C.
- Learn how to pass values and arrays to functions.
- Understand pointers and their implementation.
- Understand and implement pointers in arrays and functions.

## INTRODUCTION

In this lab, we cover the following topics:

1. Introduction to Functions  
2. Types of Functions in C Programming  
3. Introduction to Pointers  
4. 2D Pointers  
5. Void Pointer  
6. Function Pointers in C
